# Live Demo

Experience Zeno in action.

**[Launch Live Demo](https://zeno.js.org/demo/)**

## What to expect

1.  **Standard Mode**: You will see a checkbox widget. Clicking it triggers the PoW solver.
2.  **Visual Feedback**:
    *   **"Verifying..."**: The Web Worker is performing the Memory (Cuckatoo) and Time (VDF) calculations.
    *   **"Verified"**: The proof was verified by the server.
3.  **Playground**: The demo includes a styling field to test CSS customizations in real-time.
